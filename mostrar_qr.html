🔐 Configurar 2FA

Paso 1: Escanea este código QR

O copia manualmente este código:
Instrucciones:
  1. Descarga una app autenticadora: Google Authenticator, Microsoft Authenticator o Authy
  2. Abre la app y toca "+" o "Agregar"
  3. Selecciona "Escanear código QR" y apunta a la imagen arriba
  4. O copia manualmente el código gris si no puedes escanear
  5. Recibirás un código de 6 dígitos que cambia cada 30 segundos
  6. Copia ese código y confirma la activación de 2FA
